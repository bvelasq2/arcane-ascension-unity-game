# Arcane Ascension (Unity Game)

## Overview
Arcane Ascension is a Unity-based game project developed in C#. The project focuses on gameplay systems, asset integration, and object-oriented design.

## Features
- Player movement and interaction systems
- Scene and game object management
- Modular C# gameplay scripts

## Tech Stack
- Engine: Unity
- Language: C#
- Tools: Git, GitHub

## What I Learned
- Structuring a medium-sized Unity project
- Writing maintainable gameplay code
- Debugging and iterating on interactive systems
